[UVa][10019]Funny Encryption Method

－問題文：

歷史：
一個 ITESM Campus Monterrey的學生利用一個新的加密方法來玩弄數字。
這些方法由以下的步驟來組成的：

步驟： 例如
1)讀一個數字N使之加密 M=265
2)解釋N為一個十進位的數字X1 = 265(10進制)
3)轉換這個N的十進位為它的二進位的表示X1 = 100001001 (2進制)
4)讓b1等於這個二進位數字當中1的數目B1 = 3
5)解釋N為一個十六進位的數字 X2 = 265(16進制)
6)轉換這個N的十六進位為它的二進位表示X2 = 1001100101
7)讓b2等於後面這個二進位數字當中1的數目B2 = 5
8)加密的結果為 M xor (b1*b2) M xor (3*5) = 262

這個學生對計算機組織沒輒，這就是為什麼這位學生去問 ITESM Campus Monterrey
internal ACM programming Contest的裁判們關於這些數字的這兩個表示的1的bits，這
樣他可以繼續的玩弄。

任務：
你必須寫個程式，讓它讀一個數字然後給出b1和b2這兩個數字。

－輸入說明：

第一行將會包含一個數字N，代表你有幾組測資要做處理。接下來的N行(0 <=1000)當中
的每一行將會包含一個數字M(0 <=9999,用十進位表示)，這個數字就是這位學生想要加
密的數字。

－輸出說明：

你必須輸出N行，每一行包含題目所要求要的數字b1和b2，利用一個空白將這兩個數字隔
開，並且對應在要加密的數字的那行。

－範例輸入：

3
265
111
1234

範例輸出：

3 5
6 3
5 5

－思考：
進位轉換照做即可得解。
( 輸入時可善加利用setbase() )

－秒數：
0.012

－程式碼： http://nopaste.gamedev.pl/?id=6977
#include<iostream>
#include<iomanip> 
#include<sstream> 
using  namespace std;
int main ( )
{
     int N;
     while (  cin >> N  )
     {
         for (  int i =  0 ; i < N ; i++  )
         {
            string M;
             cin >> M;
            stringstream Ms;
            Ms. clear ( );
            Ms. str (M );
             int X1 [ 1000 ] =  { 0 }, X2 [ 1000 ] =  { 0 }; 
            Ms >> X1 [ 0 ];
            Ms. clear ( );
            Ms. str (M );
            Ms >> setbase ( 16 ) >> X2 [ 0 ];
             int b1 =  0, b2 =  0; 
             for (  int i =  0 ; X1 [i ] ; i++  )
             { 
                X1 [i+ 1 ] = X1 [i ] /  2;
                X1 [i ] %=  2;  
                 if ( X1 [i ]  )
                    b1++; 
             } 
             for (  int i =  0 ; X2 [i ] ; i++  )
             { 
                X2 [i+ 1 ] = X2 [i ] /  2;
                X2 [i ] %=  2;
                 if ( X2 [i ]  )
                    b2++;  
             } 
             cout << b1 <<  ' ' << b2 << endl; 
         }
     } 
     return  0;
}
